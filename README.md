# 🎓 캠플(CAMPl) - 캠퍼스 라이프 통합 플랫폼

**캠플(CAMPL)**은 대학생을 위한 통합형 캠퍼스 플랫폼입니다. 학사일정, 강의 정보, 캠퍼스 지도, 위치 기반 추천 등 다양한 정보를 제공하여 캠퍼스 생활을 더욱 편리하게 만들어주는 모바일 중심 서비스입니다.

---

## 🚀 주요 기능 개요

### 🧾 회원가입 & 로그인

- **회원가입 단계**
  1. 정보 입력 및 약관 동의
  2. 이메일 인증
  3. 대학 정보(학교, 학과, 학번 등) 등록
  4. 가입 완료 및 로그인 계정 중복 확인

- **로그인 방식**
  - 일반 로그인: 아이디 / 비밀번호 입력
  - SNS 간편 로그인: 카카오 지원

- **비밀번호 찾기**
  1. 인증 코드 기반 이메일 전송 → 인증 완료
  2. 새 비밀번호 재설정 (6자리 숫자)

- **아이디 찾기**
  1. 이메일 인증 → 아이디 자동 안내

> ※ 카카오 계정으로도 회원가입/로그인 가능

---

## 📚 튜토리얼 (최초 로그인 시)

- 사용자 맞춤 정보 설정 가이드 제공
- 주요 기능 소개 후 홈 화면으로 이동

---

## 🏠 홈

- **캠퍼스 지도**
  - 오늘의 강의 리스트
  - 오늘의 일정 리스트

- **주요 학사 공지**
  - 학교 공지사항 요약
  - 한서포탈 바로가기 등 연동

> 📌 개인 맞춤형 대시보드 형태로 정보 제공

---

## 🗺️ 맵

- 캠퍼스 주요 장소 지도
- 오늘의 강의 목록
- 오늘의 일정 목록
- 주변 장소 추천 (위치 기반)

---

## 📅 캘린더

- **학사일정 자동 등록**
- **개인일정 관리**
  - 일정 추가, 삭제, 수정 (시험, 과제, 팀플 등)
- **내 시간표 자동 등록**
  - 수강 시간 기반 자동 정리
  - 날짜별 메모 기능 포함

---

## 👤 마이페이지

1. **수강 신청 목록 (시간표)**
   - 강의명, 시간, 과목정보
2. **탈퇴 / 로그아웃**
3. **강의평**
   - 내가 작성한 강의평 목록 확인

---

## 💡 특이사항 요약

- 단일 플랫폼에서 **캠퍼스 지도 + 시간표 + 일정 + 공지 + 위치 서비스**까지 통합 제공
- 사용자 맞춤형 기능 중심의 대시보드 구성
- `카카오 로그인` 및 `이메일 인증 기반 회원 시스템` 적용

---

## 📌 기술 스택

- Frontend: React Native
- Backend: Spring Boot + JWT 인증
- DB: MySQL
- OAuth: Kakao API
- 위치기반서비스: Naver Maps API
